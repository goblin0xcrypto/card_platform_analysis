## 二、判定方法

OMEGA 開包僅 3 種形態：`$48 單抽(qty1)`、`$240 五抽(qty5,單筆)`、`$480 十抽(qty10,單筆)`。BOT 自動化訊號（涵蓋三種開法）：

1. **同秒並發**：同一秒、同一地址送出多筆交易（單抽峰值 2–5、十抽峰值 2）→ 平行簽名連續 nonce，真人/前端 UI 做不到
2. **永不混用真人慣用的批量鍵組合**：單抽型 100% 只用 $48 單抽（從不用五抽/十抽鍵）；十抽型則機械式連送 $480
3. **數百筆機關槍式開包**（單一錢包最多 824 筆）
4. **多錢包共用 BNB gas 金主**（operator 撒 0.01 BNB dust）

> 五抽（qty5）已納入偵測；全期掃描**未發現任何五抽型 bot**。

## 七、結論

Renaiss OMEGA $48 卡機在 4–5 月存在 **21 個自動化刷量機器人（5+ 組 operator）**，混用單抽與十抽大量開包。但它們 **合計淨虧 ~$10,995**、命中率維持莊家 ~0.53%——**沒有破解開獎漏洞**，S 卡純粹來自開包量。最可能動機是**衝交易量/開包數以博取未來空投或積分**（莊家抽水僅 ~4.7%，刷量成本低廉）。6 月起刷量潮已完全停止。

### 補充：S 卡盲點
本報告 S 卡數以 buyback（賣回 >$200）認定。Bot 幾乎全部賣回換現金，故其 S 卡**已被完整捕捉**；惟全平台另有約 761 張「抽到 S 卡留著不賣」的收藏型玩家 S 卡未被 buyback 法計入（詳見 `../data/renaiss_kept_s_cards.json`），但**不影響本 bot 報告**（bot 不留卡）。

### 建議
1. 將這 21 個 bot 與其 operator 列入 `config.yaml` 的 exclude，避免污染真實玩家統計。
2. 把三開法統一偵測（同秒並發 + 機關槍 + bundle 連發 + CV 規律性）固化進管線，每次更新資料後自動標記新異常。
3. 若要徹底排除「開獎可預測」疑慮，反編譯 OMEGA implementation 檢查 RNG 種子來源。
